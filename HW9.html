Probability Theory: Foundations and Axioms

1. Main Interpretations of Probability

Before the modern axiomatic framework was established, probability was understood through various intuitive lenses. While useful, each approach suffers from logical limitations.

A. Classical Interpretation (Laplace)

Based on the Principle of Indifference. If an experiment has $N$ mutually exclusive and equally likely outcomes, and if $N_A$ of these outcomes result in event $A$, then:

$$P(A) = \frac{N_A}{N}$$
Critique: It is circular. The definition relies on the term "equally likely," which essentially means "equal probability," the very concept we are trying to define. It also fails for infinite sample spaces.

B. Frequentist Interpretation (Von Mises)

Probability is defined as the limit of the relative frequency of an event as the number of trials approaches infinity.

$$P(A) = \lim_{n \to \infty} \frac{n_A}{n}$$
Critique: It requires the assumption of infinite repeatability under identical conditions, which is physically impossible. It cannot assign probability to single, non-repeatable events (e.g., "The probability of a specific candidate winning an election").

C. Subjective (Bayesian) Interpretation

Probability is a measure of an individual's degree of belief in the occurrence of an event, based on available information. It is updated as new evidence is acquired via Bayes' Theorem.

Critique: It lacks objectivity. Two rational agents with different priors may assign different probabilities to the same event.

2. The Axiomatic Approach & Measure Theory

To resolve the inconsistencies of earlier interpretations, Andrey Kolmogorov (1933) formalized probability using Measure Theory. In this framework, probability is not defined by *what it is*, but by *how it behaves*.

A probability space is a triple $(\Omega, \mathcal{F}, P)$:

  • Sample Space ($\Omega$): The set of all possible elementary outcomes.
  • Sigma-Algebra ($\mathcal{F}$): A collection of subsets of $\Omega$ (events) to which we can assign a probability. It must satisfy:
    • $\Omega \in \mathcal{F}$
    • Closed under complement ($A \in \mathcal{F} \implies A^c \in \mathcal{F}$)
    • Closed under countable unions.
  • Probability Measure ($P$): A function $P: \mathcal{F} \to [0, 1]$ satisfying the axioms below.

Random Variables as Measurable Functions

A key insight of Measure Theory is the definition of a Random Variable. It is not a variable, but a function $X: \Omega \to \mathbb{R}$.

For $X$ to be a valid random variable, it must be measurable. This means that for any interval (or Borel set $B$) in the real line, its pre-image must be an event in our Sigma-Algebra:

$$X^{-1}(B) = \{ \omega \in \Omega : X(\omega) \in B \} \in \mathcal{F}$$

This ensures that questions like "What is the probability that $X$ is greater than 5?" have a well-defined answer.

3. Derivations from the Axioms

Using Kolmogorov's axioms (Non-negativity, Normalization, and Countable Additivity), we can derive all other properties of probability.

Property 1: Subadditivity (Boole's Inequality)

Statement: For any sequence of events $A_1, A_2, \dots$ (not necessarily disjoint):

$$P\left(\bigcup_{i=1}^{\infty} A_i\right) \leq \sum_{i=1}^{\infty} P(A_i)$$
Proof:
1. Construct a sequence of disjoint events $B_i$ such that $B_1 = A_1$, $B_2 = A_2 \setminus A_1$, $B_3 = A_3 \setminus (A_1 \cup A_2)$, and so on.
2. By construction, $B_i \subseteq A_i$ and $\bigcup B_i = \bigcup A_i$.
3. Since $B_i$ are disjoint, we apply the Axiom of Countable Additivity:
$$P\left(\bigcup_{i=1}^{\infty} A_i\right) = P\left(\bigcup_{i=1}^{\infty} B_i\right) = \sum_{i=1}^{\infty} P(B_i)$$ 4. By the monotonicity of probability (since $B_i \subseteq A_i$, then $P(B_i) \leq P(A_i)$), we obtain:
$$\sum_{i=1}^{\infty} P(B_i) \leq \sum_{i=1}^{\infty} P(A_i)$$ Q.E.D.

Statement: For two events $A$ and $B$, the probability of their union is:

$$P(A \cup B) = P(A) + P(B) - P(A \cap B)$$
Proof:
1. We can decompose the union $A \cup B$ into three disjoint sets:
$$A \cup B = (A \setminus B) \cup (A \cap B) \cup (B \setminus A)$$ 2. By additivity:
$$P(A \cup B) = P(A \setminus B) + P(A \cap B) + P(B \setminus A)$$ 3. Note that event $A$ is the disjoint union of $(A \setminus B)$ and $(A \cap B)$. Thus, $P(A) = P(A \setminus B) + P(A \cap B)$, which implies $P(A \setminus B) = P(A) - P(A \cap B)$. Similarly for $B$.
4. Substituting these back into step 2:
$$P(A \cup B) = [P(A) - P(A \cap B)] + P(A \cap B) + [P(B) - P(A \cap B)]$$ 5. Canceling terms yields:
$$P(A \cup B) = P(A) + P(B) - P(A \cap B)$$ Q.E.D.
